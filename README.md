EasyAPI静态资源库
--------

## 环境准备：
Node.js 
- Mac安装Node.js
- Windows安装Node.js

运行sudo node start.js启动使用80端口的静态资源服务

## 包说明
>images 图片文件  
>styles 样式表  
>scripts 脚本文件  
>libs 第三方库  
>bootstrap   
>select2   
>...

说明：第三方引用，必须放置在根目录

映射域名static.easyapi.com，使用[SwitchHosts](http://oldj.github.io/SwitchHosts/)，设置127.0.0.1 static.easyapi.com。
